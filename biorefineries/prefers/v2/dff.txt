### Summary: Python Implementation Structure

Here is how you structure the "Backward Sizer" for TFF in your TEA code.

```python
import numpy as np

class TFF_Sizer:
    def __init__(self, flux_LMH, rejection_imp=0.1, rejection_target=1.0):
        self.J = flux_LMH
        self.sigma_imp = rejection_imp
        self.sigma_target = rejection_target

    def backward_design(self, vol_feed_L, target_purity_removal, target_concentration_factor):
        """
        target_purity_removal: Fraction of impurity to REMOVE (e.g. 0.99 means 1% left)
        """
        
        # 1. Concentration Step (Volume Reduction)
        # Calculate Volume to remove to hit VCF
        vol_retentate_L = vol_feed_L / target_concentration_factor
        vol_permeate_conc = vol_feed_L - vol_retentate_L
        
        # 2. Diafiltration Step (Washing)
        # Calculate Diavolumes (N) needed to hit purity target
        # Formula: N = ln(C_in / C_out) / (1 - sigma)
        # Note: target_purity_removal = 1 - (C_out/C_in)
        fraction_remaining = 1 - target_purity_removal
        
        # Handle edge case where sigma is close to 1 (impossible to wash out)
        if self.sigma_imp >= 0.99:
             N_dv = 999 # Error state
        else:
             N_dv = np.log(1 / fraction_remaining) / (1 - self.sigma_imp)
        
        vol_buffer_L = N_dv * vol_retentate_L
        
        # 3. Sizing (Area)
        # Total Permeate to push through membrane
        total_vol_permeate = vol_permeate_conc + vol_buffer_L
        
        # Assume 4 hour process time constraint
        process_time_h = 4.0 
        area_m2 = total_vol_permeate / (self.J * process_time_h)
        
        return {
            "Membrane_Area_m2": area_m2,
            "Buffer_Usage_L": vol_buffer_L,
            "Total_Permeate_L": total_vol_permeate,
            "Diavolumes": N_dv
        }

# Example
tff = TFF_Sizer(flux_LMH=50, rejection_imp=0.05) # 50 LMH design flux, salts pass freely
result = tff.backward_design(vol_feed_L=1000, target_purity_removal=0.99, target_concentration_factor=5.0)

print(f"Area Needed: {result['Membrane_Area_m2']:.2f} m2")
print(f"Buffer Consumed: {result['Buffer_Usage_L']:.2f} L")

```

* **Chromatography Cost Driver:** **Resin Volume** (Cycle time doesn't change cost much, just throughput).
* **Filtration Cost Driver:** **Membrane Area** (Flux is the king parameter here).
